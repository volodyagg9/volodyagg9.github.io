Каждый пост — это один `.md` файл в `src/content/posts/`. Имя файла становится
адресом: `remnawave-nodes.md` → `/posts/remnawave-nodes/`.

Сверху — frontmatter (шапка между `---`):

```yaml
---
title: Заголовок поста
description: Короткое описание для превью и RSS
date: 2026-06-01
tags: [vpn, python]
draft: false   # true — не публиковать, спрятать из списка
---
```

Дальше — обычный Markdown. Работает всё, к чему ты привык в Obsidian:
заголовки, **жирный**, списки, `код`, цитаты, картинки.

> Кинул файл — запушил — Cloudflare сам пересобрал. Никакой возни.

Копируешь свои заметки из Obsidian, чистишь вики-ссылки — и готово.
